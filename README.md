# 银山打字通 (推箱子游戏)

这是一个结合了英语语法问答和经典推箱子玩法的网页游戏。玩家需要通过正确回答英语语法题目来控制角色移动，将所有箱子推到指定的目标点即可过关。

## 如何运行

直接在现代浏览器（如 Chrome, Firefox, Edge 等）中打开 `index.html` 文件即可开始游戏。

## 玩法说明

- **移动**: 回答左侧的英语题目，输入正确的单词答案后，角色会向当前朝向移动一格。
- **转向**: 在输入框中输入 `up`, `down`, `left`, `right` 可以改变角色的朝向。
- **目标**: 将所有箱子（`box.png`）推到目标点（`ball.png`）上。

## 文件说明

- **`index.html`**: 游戏的主入口 HTML 文件。它创建了 `<canvas>` 元素用于绘制游戏，并引入了所有必要的 CSS 和 JavaScript 文件。

- **`css/style.css`**: 为游戏页面提供基本的 CSS 样式，例如将游戏画布居中显示。

- **`js/script.js`**: 此文件负责初始化游戏。它获取 `<canvas>` 元素，加载所有图片资源，并设置键盘输入和鼠标点击的事件监听器来捕捉玩家的操作。

- **`js/function.js`**: 这是游戏的核心逻辑所在。它包含了处理关卡初始化、绘制地图、响应玩家按键、处理答案对错、角色移动、游戏胜利条件判断等功能的函数。

- **`js/mapdata.js`**: (推测) 定义并存储了游戏中所有关卡的布局。每个关卡通常由一个二维数组表示，数组中的数字对应不同的游戏元素（如墙、路、箱子、目标点等）。

- **`js/worddata.js`**: (推测) 包含了游戏中的所有英语语法问题。每个问题对象可能包含题目文本、正确答案、提示等信息。

- **`image/`**: 这个目录存放了游戏渲染所需的所有视觉资源。
